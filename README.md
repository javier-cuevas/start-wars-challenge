# Star Wars Challenge v1

Technical exercise.

Project generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.0.6.

## Dependencies

```Angular CLI: 11.0.6```
```Node: ^12.17.0```

## Installation

Run `npm install`

## Development server

Run `ng serve --ssl`
Navigate to ```http://localhost:4200/```

## Build Production

Run `ng b --prod`
